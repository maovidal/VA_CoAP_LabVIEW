# VA CoAP LabVIEW

CoAP library for LabVIEW

**NOTE**
This is a work in progress that can serve as a reference to implement CoAP in LabVIEW.
This is not complete. Pull requests are welcome.

The CoAP specification can be found here: https://www.rfc-editor.org/rfc/rfc7252


## Requirements

- LabVIEW 2020

Installed with JKI VI Package Manager:

- JSON API VIPM Communitty
- NI Unicode Tools
